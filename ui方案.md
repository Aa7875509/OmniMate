### 设计理念与视觉风格

本方案调整为 **“ChatGPT 官方风格”**：克制、清晰、低装饰，优先保证内容可读性与输入效率。

- **色彩主题**：使用中性浅色背景（如 `#f7f7f8` / `#ffffff`），搭配浅灰边框（如 `#e5e7eb`）。主色保留青色 `#2dd4bf`，仅用于焦点、主按钮和关键状态。
- **材质与质感**：不使用毛玻璃、渐变和大面积光效。卡片统一为纯白或浅灰纯色背景，边界由 1px 细边框定义。
- **字体系统**：保持无衬线体系（Inter / system-ui），字号层级简洁：标题、正文、辅助文本三档。
- **动效策略**：只保留必要过渡（悬停、聚焦、加载），动画时长短、幅度小，避免视觉噪音。

### 主界面布局方案

主界面改为 **“内容优先”** 布局：对话与输入是核心，视觉装饰退到次要位置。

#### 整体结构

应用窗口分为三个区域：

1. **顶部栏（Header）**：品牌、模型状态、全局设置。
2. **主内容区（Content）**：左侧实时对话浮层 + 中央舞台区（可承载数字人/可视化）。
3. **底部输入区（Composer）**：固定输入栏，支持文本与语音入口。

```
+---------------------------------------------------------------+
| [Logo] [模型状态]                           [设置]            |
|---------------------------------------------------------------|
| [实时对话浮层]            [主舞台区 / 数字人区]              |
|                                                               |
|                                                               |
|---------------------------------------------------------------|
| [语音] [输入框............................] [发送] [历史]      |
+---------------------------------------------------------------+
```

#### 顶部栏（Header）

- **左侧**：OmniMate 品牌标识。
- **中间**：当前模型标签 + 连接状态（在线 / 思考中 / 离线）。
- **右侧**：设置图标按钮（进入模型管理中心）。
- **样式规范**：
  - 高度约 56-64px；
  - 背景纯白；
  - 1px 浅边框；
  - 轻阴影（可选）。

#### 实时对话区（Chat Panel）

- 固定在左侧浮动显示，支持内部滚动。
- 无默认示例消息，首次进入显示空状态提示（如“开始一段新对话”）。
- 消息规则：
  - 用户消息右对齐；
  - 助手消息左对齐；
  - 气泡圆角统一，弱色块区分角色（避免高饱和）。
- 支持 Markdown 渲染和长文本换行。

#### 底部输入区（Composer）

- 采用 ChatGPT 风格横向输入条：
  - 左：语音按钮（次级）
  - 中：主输入框（占满剩余空间）
  - 右：发送按钮（主操作）
  - 附：历史入口（次级文本按钮）
- 输入框视觉：大圆角、浅边框、纯色浅底（不使用渐变）。
- 状态反馈：
  - 请求中禁用输入；
  - 发送按钮显示 loading；
  - 错误以轻提示展示，不打断输入节奏。

### 关键组件规范

#### 3D 数字人视窗（可选主舞台）

- 作为中心信息承载区，可显示数字人、视觉分析结果或占位内容。
- 默认不抢视觉焦点，优先让位给对话与输入。

#### 模型管理中心（Modal）

- 使用标准弹窗或抽屉样式，不做全屏强动效。
- 内容结构优先表单效率：
  - 模型类型
  - OpenAI / Ollama 配置
  - 主题色
  - 恢复默认 / 刷新状态 / 保存并切换

### 落地到 Ant Design Vue 的建议

- **布局**：`a-layout` / `a-layout-header` / `a-layout-content`
- **对话**：`a-card` + `a-list` + 自定义消息气泡容器
- **输入区**：`a-input` + `a-button` + `a-space-compact`
- **状态**：`a-tag` / `a-badge` / `a-alert`
- **设置**：`a-modal` + `a-form` + `a-select` + `a-input`

### 视觉约束（确保“像 ChatGPT”）

- 不使用大面积霓虹和高斯发光；
- 不使用任何渐变作为主背景；
- 边框、字号、间距优先一致性；
- 让“内容”和“输入行为”成为第一视觉中心。
